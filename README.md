# Guía de instalación

1- Pegar el proyecto en la carpeta htdocs de un servidor local (WAMP/XAMP).

2- Importar el archivo prensaglobal.sql incluido en el directorio /database a través de phpMyAdmin.

3- Configurar el archivo .env en la raíz del proyecto para ajustar las credenciales de conexión a la base de datos local.

4- Asegurar que las extensiones intl, mbstring y mysqli de PHP estén habilitadas.

5- Acceder mediante el navegador a la ruta /public del proyecto.
